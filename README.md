## Laravel

 Ini merupakan Website Persewaan Lapangan Basket dan Futsal Solo Technopark.
 Fitur:
 1. Profile Solo Technopark
 2. Halaman Login
 3. Halaman User
 4. Halaman Admin
 5. Laporan PDF admin

 Build App:
 1. PHP 7.4
 2. Tailwind 3.4.1

## Cara Install
 1. composer install
 2. npm install
 3. php artisan key:generate
 4. php artisan migrate
 5. cp .env.example .env
 6. npm run dev
 7. php artisan serve
